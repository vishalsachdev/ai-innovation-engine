## Budget Justification

### 1. Research Assistants — $40,000

**PhD Research Assistant (1 RA, 25% appointment, 2 years)**
- $16,000/year salary for 9-month academic year + summer supplement
- Recruited from Business Administration, Information Sciences, or Computer Science doctoral programs
- Responsibilities: platform instrumentation, data collection pipeline, qualitative coding of interaction patterns, dataset curation, and co-authorship on research papers
- Reports jointly to PI (Sachdev) and Co-I (Carrasco Kind) for technical integration with DSRS

**Graduate Research Assistants (hourly, DSRS student interns)**
- $4,000/year for 2 DSRS student interns contributing to Atlas agentic framework integration and data pipeline development
- These students are existing DSRS interns; grant funding supplements their hours for project-specific tasks
- Supervised by Co-I Carrasco Kind

### 2. Fringe Benefits — $10,000

- Graduate assistant fringe at university rate (~25% of salary)
- Applied to PhD RA and DSRS intern compensation

### 3. LLM API & Compute Credits — $8,000

**LLM API Credits ($4,000 Year 1, $2,000 Year 2)**
- API credits for student-facing AI environments in instrumented courses (OpenAI, Anthropic, Google research tier)
- Year 2 costs decline as DSRS self-hosted LLM endpoints (A100 GPUs) absorb more inference load

**Compute Supplement ($1,000/year)**
- Supplemental cloud compute for burst capacity during data collection peaks (fall/spring semesters)
- Note: Core compute is provided by DSRS at no cost (486 CPU cores, A100 GPUs, $160K+ Azure credits)

### 4. Data Collection & IRB — $5,000

**IRB Protocol Development ($3,000 Year 1)**
- Consultant support for informed consent procedures adapted to educational platform research
- Legal review of data retention, de-identification, and cross-course data sharing protocols

**Data Curation ($1,000/year)**
- De-identification pipeline validation
- Annotation scheme development and inter-rater reliability testing
- Dataset documentation following FAIR data principles

### 5. Research Dissemination — $8,000

**Conference Travel ($3,000 Year 1, $4,000 Year 2)**
- Year 1: 1 conference (ICIS or AMCIS) — present preliminary findings
- Year 2: 2 conferences (ICIS + AOM or domain venue) — present full papers from RQ1/RQ2

**Open Access Fees ($1,000 Year 2)**
- Article processing charges for one open-access publication

### 6. Participant Compensation — $4,000

**Interview Participants ($1,000/year)**
- 10-15 semi-structured interviews with faculty and administrators per year
- $50 gift cards per participant

**Finance Domain Experiment Participants ($1,000/year)**
- Supplemental compensation for students completing Zhang's finance AI decision tasks outside of class assignments
- Administered through Gies Business Research Lab protocols

## Cost Savings from Existing Infrastructure

This proposal achieves significant cost savings by leveraging existing Gies investments rather than building new systems:

| Asset | Estimated Replacement Cost | Provided By |
|---|---|---|
| DSRS compute cluster (486 cores, A100 GPUs) | $200,000+ | DSRS (Carrasco Kind) |
| Microsoft Azure credits | $160,000 | DSRS Azure Award |
| Atlas agentic framework | $50,000+ dev cost | DSRS |
| Canvas MCP (90+ tools, PyPI-published) | $100,000+ dev cost | Sachdev (open-source) |
| Student intern labor (DSRS) | $30,000/year | DSRS existing budget |
| **Total leveraged value** | **$540,000+** | |

The $75,000 grant request unlocks research capability built on over $540,000 in existing infrastructure — a 7:1 leverage ratio.

## Notes

- **No administrative or coordinator positions** are requested — project coordination is handled by the three PIs within their existing appointments.
- **No faculty salary** is requested — all three investigators contribute effort through existing appointments.
- **No new infrastructure** is built — the project integrates Canvas MCP and DSRS systems that are already operational.
- **Fringe benefits** are included at the university's current graduate assistant rate (~25%).
- **No internal cost recovery overhead** is included per CFP guidelines.
- All spending completes before end of AY 2027-28 per proposed timeline.
