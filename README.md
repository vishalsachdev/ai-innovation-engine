# Gies AI Innovation Engine

Research and planning repository for building a multi-sided AI innovation platform at Gies College of Business, UIUC.

## What Is This?

A comprehensive research effort exploring how open-source AI tools (gstack, Paperclip, Hermes Agent, ResearchChatAI) can be combined with existing Gies infrastructure (Magelli Office, Origin Ventures, ISOI, Canvas MCP) to create a self-sustaining innovation engine where:

- Faculty, staff, corporates, and alumni post real-world AI challenges
- 9,000+ students build solutions using shared infrastructure
- Every project grows a compound skill library
- Research data flows to faculty studying human-AI collaboration
- Viable tools get commercialized through Origin Ventures
